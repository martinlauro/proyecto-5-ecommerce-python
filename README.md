# proyecto-5-ecommerce-python
Análisis de ventas ecommerce con Python y visualizaciones



Análisis completo de datos de ventas ecommerce usando Python puro,
sin base de datos SQL.

## Tecnologías
- Python (pandas, matplotlib, seaborn)

## Contenido
- `analisis_ecommerce.py` — Script de análisis y visualización
- `dashboard_ecommerce.png` — Dashboard con 4 gráficos exportado automáticamente

## Análisis realizados
- Resumen general de ventas
- Ingresos por categoría
- Top 5 productos más vendidos
- Evolución mensual de ventas
- Distribución por país
